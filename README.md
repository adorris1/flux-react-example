# Flux-React-Example

> This is a Work In Progress

An isomorphic flux-react example application using fluxible.

## Setup

```bash
$ npm install
```


## Run the app

```bash
$ npm run build
$ npm start
```

Open http://localhost:3000


## Run in dev mode

```bash
$ npm run dev
```

This will use `nodemon` and `webpack` to watch for changes and restart and
rebuild as needed.

Open http://localhost:3000
